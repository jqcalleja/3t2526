This is how a basic webpage looks like. This uses some of the basic tags used in webpage development. The basic HTML tage are:

  • div - divsion tag
  • p - paragraph tag
  • a - anchor tag
  • img - image tag
  • ul - unordered list tag
  • ol - ordered list tag
Some tags are usually used to organize the contents of our page. These are called the semantic tags. Here are the 6 semantic tags in HTML:
  1. header
  2. nav
  3. main
  4. section
  5. article
  6. footer
